There are 3 Hidden plots on the page. Toggle the visibility of each one with the buttons below.

Each plot has it's height and width set by a different mechanism. The first plot uses the custom html attributes "data-width" and "data-height". The second passes in a "width" and "height" option to the plot constructor. The third uses the jqplot wide config default settings, but overrides them to 200px high by 600px wide.

The default config settings (if not overridden) are 300px wide by 400px high. The config default settings will apply to all plots on a page. The config setting has the lowest order of precedence, however. jqPlot will first look for a setting in the options object passed to the plot constructor. It will then look for the custom html attributes. If these are not found, it will use the config defaults.

Note, if the plot target generates a containing box (display property is not "none") and has a height/width css specification, jqPlot will use those values. These additional mechanisms for sizing the plot are effective only if the plot target or one of it's parent containers is initially set to "none" or no css height/width values are provided for the plot target.
